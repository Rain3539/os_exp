# 从零构建操作系统 - 实验2

这是一个基于RISC-V架构的最小化操作系统内核。
该项目是《从零构建操作系统-学⽣指导⼿册》中实验2的实现。



## 如何编译和运行

### 前提条件

你需要安装好实验0中要求的所有开发工具，包括：
- `make`
- `qemu-system-riscv64`
- `riscv64-unknown-elf-gcc` 工具链

### 步骤

1.  **编译内核**
    在项目根目录下运行`make`命令：
    ```bash
    make
    ```
    这将会编译所有源文件并链接成一个名为 `kernel.elf` 的可执行文件。

2.  **在QEMU中运行**
    使用以下命令启动QEMU并加载你的内核：
    ```bash
    make qemu
    ```
    你将会在终端看到输出：
    ```
    Hello OS 
    ```

3.  **退出QEMU**
    按下 `Ctrl + A`，然后松开，再按下 `X` 键。

4.  **清理生成文件**
    运行以下命令删除所有编译生成的文件：
    ```bash
    make clean
    ```
